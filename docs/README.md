## ✨ 功能特性

### 🎯 核心功能
- **📁 多平台网盘支持** - 支持多种主流网盘平台
- **🔍 公开API** - 支持API数据录入，资源搜索
- **🏷️ 自动预处理** - 系统自动处理资源，对数据进行有效性判断
- **📊 自动转存分享** - 有效资源，如果属于支持类型将自动转存分享
- **📱 多账号管理** - 同平台支持多账号管理
